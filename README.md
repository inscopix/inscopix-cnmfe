# isx-cnmfe
Algorithm for automated source extraction from 1P data

Constrained non-negative matrix factorization for microendoscopic data


Required dependencies:
- armadillo
- Intel_MKL (or LAPACK/BLAS)
- OpenCV
- mlpack
- mio
- ThreadPool

Optional dependencies:
- cvxcore
- scs
- hdf5
- libtiff
- Catch
